# Feature 3: update main page UI

## Requirement
- main page UI is limited to a fixed container width 700px
- it consist of 3 parts:
  > top: left handside is the the page header/title, right handside is a action button
  > bottom: a navigation bar with function
    > transactions: show a list of expenses for each month
    > statistics: show some charting base on the expenses
    > meta: setup the application meta data for app usage
    > transfer: calculate the money required to pay back for each month
- you don't need to create the content of each pages, just create the UI layout with some dummy text and images
